# Claude Code Instructions

## Package Manager

Always use **pnpm** instead of npm or yarn.

```bash
# Install dependencies
pnpm install

# Add a package
pnpm add <package>

# Add a dev dependency
pnpm add -D <package>

# Run scripts
pnpm dev
pnpm build
```
## Tables

Always use tanstack/react-table when creating tables

## CSS
Always use tailwind
